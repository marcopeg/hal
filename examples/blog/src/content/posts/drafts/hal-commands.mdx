HAL's built-in commands (`/start`, `/help`, `/model`, `/engine`, `/clear`) cover the basics. But the real unlock is writing your own.

---

## What a custom command looks like

Drop a `.mjs` file into `.hal/commands/` in your project:

```js
// .hal/commands/standup.mjs
export default {
  description: "Generate a standup summary from recent git commits",
  prompt: `
    Look at the last 24 hours of git commits in this project.
    Write a short standup update in this format:
    - What I did yesterday
    - What I'm doing today
    - Any blockers
    Keep it under 5 bullets total.
  `,
};
```

Now `/standup` is a real Telegram command. Send it, and Claude runs the prompt against your project.
