# `/clear`

Resets the active LLM session for the user while preserving user files such as uploads and downloads.

## Defaults

- `enabled: true`
- `showInMenu: true`
- `showInHelp: true`

## Config

Supported keys:

- `enabled`
- `showInMenu`
- `showInHelp`
- `message`

The custom message changes only what the user sees after the reset.

See [Built-in commands configuration](../../../config/commands/README.md).
