# `/help`

Shows HAL's formatted command list, typically based on `${HAL_COMMANDS}`.

## Defaults

- `enabled: true`
- `showInMenu: true`
- `showInHelp: true`

## Config

Supported keys:

- `enabled`
- `showInMenu`
- `showInHelp`
- `message`

The default message is `${HAL_COMMANDS}`.

See [Built-in commands configuration](../../../config/commands/README.md).
