# 057 — Status/info command with feature flags

**Status**: draft

## Description
Add a status/info command as a system HAL command that is enabled by default. When the command runs, it should show:
- the name of the project
- the CWD
- the current engine and model (for the engine show the model, but not the offer)
- the session configuration
- the current context that is given to the project

In the options for this command, it should be enabled by default but can be disabled. Add feature flags so each component can be disabled:
- a feature flag for the CWD (true/false)
- a feature flag for the engine and model (true/false)
- a feature flag for the session (true/false)
- a feature flag for the context (true/false)

All feature flags default to true so the info can be executed but it doesn’t show components that are disabled.
