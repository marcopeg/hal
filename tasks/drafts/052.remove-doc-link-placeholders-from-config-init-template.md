# 052 — Remove doc link placeholders from config init template

**Status:** draft

## Summary

The `src/init-template.yaml` file contains placeholder tokens (e.g. `{{HAL_DOCS_CONFIG}}`, `{{HAL_DOCS_TELEGRAM}}`, `{{HAL_DOCS_TELEGRAM_CREATE_BOT}}`) for documentation links that are currently unresolved at init time. Remove these placeholders and replace them with hardcoded comment text directly in the template file.

## Context

The init template is used to generate a starter `hal.config.yaml` for new users. The doc link placeholders are never substituted at runtime — documentation references should instead be written directly as static comments in the template.
