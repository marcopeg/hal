HAL skills now fit Telegram better.

Run Claude Code, Copilot, and Codex from your phone. HAL turns Telegram into a remote control for AI coding agents, and v1.7.0 cleans up two important edges: skills marked `telegram: true` now plug more cleanly into slash-command workflows, and fragmented messages are handled more reliably.

If you use HAL away from the keyboard, this is a small release with real UX payoff.
